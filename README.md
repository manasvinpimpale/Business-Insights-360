AtliQ Hardware is a rapidly growing company that decided to adopt Power BI for data analytics to gain a competitive edge and make informed business decisions. This project focuses on delivering insights across key business areas including Finance, Sales, Marketing, and Supply Chain.

I, Manasvi, completed this project by following the Codebasics Power BI Course, applying real-world business scenarios and industry best practices.

## Tech Stack

1. SQL
2. Power BI Desktop
3. Excel
4. DAX Language
5. DAX Studio (for report optimization)
6. Project Charter File

## Power BI Concepts & Techniques Applied

1. Asking the right business questions before starting the analysis
2. Creating calculated columns and measures using DAX
3. Data modeling and relationship management
4. Using bookmarks to switch between visuals
5. Page navigation using buttons
6. Handling divide-by-zero errors using the DIVIDE() function
7. Creating dynamic titles based on applied filters
8. KPI indicators and performance tracking
9. Conditional formatting using icons and background colors
10. Data validation techniques
11. Publishing reports to Power BI Service
12. Setting up a personal gateway for automatic data refresh
13. Creating and managing Power BI Apps

## Company Background

AtliQ Hardware sells computers and accessories through retailers and direct distributors worldwide. The company recently experienced unexpected losses in its American market due to decision-making based on surveys, intuition, and limited Excel analysis. To improve performance and remain competitive, AtliQ Hardware decided to build a strong analytics foundation using Power BI.

## Project Outcome 

This project provides a clear analytical framework to support data-driven decision-making. The Power BI reports generated help answer critical business questions, track performance, and uncover insights that enable AtliQ Hardware to improve efficiency, reduce losses, and strengthen its market position.
